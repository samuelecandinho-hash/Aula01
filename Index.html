20 Tags HTML(titulo)

Começar

A tag p serve para definir parágrafos(paragrafo)

A tag h1 serve para definir o cabeçalho principal

A tag img serve para inserir imagens.

A tag strong serve para definir texto em negrito. A tag em serve para definir texto em itálico. A tag a serve para criar um link para outra página ou recurso.
A tag div serve para definir uma divisão ou seção em um documento.

A tag var serve para definir uma variável.

A tag cite serve para definir uma citação. A tag code serve para definir um trecho de código.
A tag blockquote serve para definir uma citação em bloco.

A tag base serve para definir a URL base para todas as URLs relativas em um documento.


A tag br serve para inserir uma quebra de linha.
A tag q serve para definir uma citação em linha. A tag del serve para definir texto que foi removido. 
A tag pre serve para definir um bloco de código pré-formatado.
A tag samp serve para definir um exemplo de saída de um programa. A tag kbd serve para definir texto que deve ser digitado pelo usuário. A tag dfn serve para definir um termo.
